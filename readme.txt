Инструкция по установке:
1) Установить Java 8 и настроить переменную JAVA_HOME
2) Установить Python 3.5 и прописать путь до него в PATH
3) Запустить gradlew build для установки грэдла и билда сборки
  3.1) При первом запуске скачается gradle нужной версии в C:\Users\%имя_пользователя%\.gradle
  3.2) Сервис билдится в виде универсального war-файла, который можно как задеплоить в любом сервлет-контейнере,
       так и запустить как standalone jar c помощью команды java -jar (в war-файл включен embedded tomcat)
4) Запустить gradlew installPyLibs для установки библиотек питона

Инструкция по запуску:
1) Запустить gradlew bootRun для запуска сервиса (запустится по адресу localhost:8080)
2) Запустить gradlew startPyGUI для запуска тестового клиентского приложения (запустится по адресу localhost:8000)

- Все настройки сервиса хранятся в serviceparams.xml
- При необходимости настроить адрес сервиса в клиентском приложении можно в файле GUI-django/conf.txt
